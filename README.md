# Hotel_Management.sql
The Hotel Management System is a SQL-based database project designed to manage hotel operations such as customer details, room availability, and bookings.  
This project demonstrates core SQL skills required for a Data Analyst / Database role, including:  Database design  Table relationships  Joins and filtering  Data analysis queries

🎯 Objectives

Manage customer information

Track room availability

Handle room bookings

Generate useful business insights using SQL

🛠️ Tech Stack

Database: MySQL

Language: SQL

Tools: MySQL Workbench
